# Final-Year-Project 2022
Final-Year Project: Performance Appraisal and Reporting Tool. 
Developed a web-based performance appraisal system using HTML, CSS, JavaScript, and PHP to streamline employee evaluations and reporting processes.
Repository Description
The Final-Year-Project repository showcases a complete performance appraisal system designed to help organizations manage employee evaluations more efficiently. 
The tool integrates multiple modules for login, dashboards, target tracking, and reporting, with a structured database backend.

Key Features
- User & Admin Dashboards: Separate interfaces for employees and administrators to manage performance data.
- Login & Registration Modules: Secure authentication for users and faculty profiles.
- Target Tracking: Pages for achieved/reached targets, with reporting views for administrators.
- Forms & Input Handling: Multiple PHP forms for capturing appraisal details and performance metrics.
- Data Visualization: Integration of charts for graphical representation of performance data.
- Database Integration: Includes presidency_university.sql for structured storage of appraisal records.
- Responsive UI: HTML/CSS-based design with navigation pages and portfolio-style layouts.

Repository Structure Highlights
- index.php → Main entry point of the application.
- a_login.php, u_login.php, u_reg.php → Authentication and registration modules.
- a_dashboard.php, u_dashboard.php, f_dash.php → Dashboards for different user roles.
- form_*.php → Multiple appraisal forms for capturing performance data.
- admin_view_* → Admin-specific views for monitoring targets and faculty performance.
- presidency_university.sql → Database schema for storing appraisal data.
- style.css → Core stylesheet for UI design.

Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: PHP
- Database: MySQL (via included SQL schema)

- Purpose
This project was developed to digitize and streamline performance appraisal processes, reducing manual paperwork and enabling faster reporting.
It demonstrates skills in web development, database design, and user interface creation, making it a strong portfolio piece for IT and analyst roles.

Note: Download and install XAMPP, an open-source, cross-platform web server solution stack developed by Apache, to run this project.
